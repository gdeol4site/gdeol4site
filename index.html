STONED SELFIES  
This app was built using code from the paper Beyond Generative Models: Superfast Traversal, Optimization, Novelty, Exploration and Discovery (STONED) Algorithm for Molecules using SELFIES. A starting molecular structure is chosen and the genetic algorithm is capable of producing novel molecules on par with GPU powered deep learning models without the expensive resources. The app hosted using streamlit.
